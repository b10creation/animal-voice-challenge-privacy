利用規約

どうぶつモノマネ選手権

最終更新日: 2026年2月7日

本利用規約（以下「本規約」）は、「どうぶつモノマネ選手権」（以下「本アプリ」）の利用条件を定めるものです。ユーザーの皆様には、本規約に同意いただいた上で、本アプリをご利用いただきます。

第1条（適用）

本規約は、ユーザーと本アプリの運営者（以下「運営者」）との間の本アプリの利用に関わる一切の関係に適用されます。

第2条（サービス内容）

本アプリは、以下のサービスを提供します：

  • 動物の鳴き真似を録音し、AIが類似度を採点する機能
  • 全国ランキングおよび動物別ランキングの表示
  • 採点結果のSNSシェア機能
  • プロフィール管理および挑戦履歴の閲覧

第3条（利用登録）

  1. 本アプリは、Googleアカウントまたは匿名ログインにより利用を開始できます。
  2. ユーザーは、正確かつ最新の情報を提供するものとします。
  3. ユーザーは、自身のアカウント情報を適切に管理する責任を負います。

第4条（利用料金）

  1. 本アプリの基本機能は無料でご利用いただけます。ただし、1日あたりの挑戦回数に制限があります。
  2. プレミアムプラン（有料）にご加入いただくと、無制限プレイ、広告非表示、限定動物の解放などの特典をご利用いただけます。
  3. 有料サービスの料金は、アプリ内に表示される金額に従います。
  4. 購入後の返金については、各アプリストア（App Store / Google Play）の規定に従います。

第5条（禁止事項）

ユーザーは、本アプリの利用にあたり、以下の行為をしてはなりません：

  • 法令または公序良俗に反する行為
  • 犯罪行為に関連する行為
  • 不正アクセス、サーバーへの過度な負荷をかける行為
  • 採点結果を不正に操作する行為、またはその試み
  • 他のユーザーの利用を妨害する行為
  • 他のユーザーに対する誹謗中傷、嫌がらせ
  • 不適切な音声（暴力的、性的、差別的な内容）の録音・投稿
  • 本アプリの運営を妨害する一切の行為
  • その他、運営者が不適切と判断する行為

第7条（知的財産権）

  1. 本アプリに関する知的財産権は、運営者または正当な権利者に帰属します。
  2. ユーザーが録音した音声データの著作権はユーザーに帰属しますが、ランキング表示およびサービス提供のために運営者が利用できるものとします。

第8条（アカウント削除）

  2. アカウント削除により、ユーザーのデータ（スコア、音声ファイル、プロフィール情報）は完全に削除されます。
  3. 削除されたデータの復元はできません。

  3. 運営者は、ユーザー間のトラブルについて一切関与せず、責任を負わないものとします。
