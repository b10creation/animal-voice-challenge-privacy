どうぶつモノマネ選手権

最終更新日: 2026年2月11日

「どうぶつモノマネ選手権」（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本アプリにおける情報の収集、使用、開示に関する方針を説明します。

1. 収集する情報

1.1 アカウント情報

  • Googleアカウントでログインした場合：表示名、メールアドレス、プロフィール画像URL
  • Appleアカウントでサインインした場合：表示名（ユーザーが共有を許可した場合）
  • 匿名ログインの場合：Firebase が生成する匿名ユーザーID
  • ユーザーが設定したニックネーム

1.2 音声データ

  • ユーザーが録音した動物の鳴き真似の音声ファイル（3秒間、AAC形式）
  • 音声ファイルは Firebase Storage に保存されます
  • AI採点時に、音声データと選択した動物の種類が Google Gemini AI に送信されます（詳細は「3. AI採点におけるデータ送信と同意」をご参照ください）

1.3 利用データ

  • 採点スコアおよびスコアの内訳
  • AIによる評価コメント
  • プレイ回数、挑戦履歴
  • ランキング情報

2. 情報の使用目的

収集した情報は以下の目的で使用します：

  • AI採点: 録音された音声を Google Gemini API に送信し、動物の鳴き真似の類似度を評価します
  • ランキング表示: 全国ランキングおよび動物別ランキングにスコアとニックネームを表示します
  • SNSシェア: ユーザーの操作により、スコアカード画像をSNSにシェアする機能を提供します
  • サービス改善: アプリの品質向上のために利用状況を分析します

3. AI採点におけるデータ送信と同意

本アプリの核となる機能であるAI採点では、外部AIサービスへのデータ送信が行われます。

3.1 送信されるデータ

AI採点を実行する際、以下のデータがサーバーに送信されます：

  • ユーザーが録音した音声データ（約3秒間のAAC音声ファイル）
  • 選択した動物の種類（例：「猫」「犬」など）

3.2 送信先

  • サービス名: Google Gemini AI（Google LLC が提供する生成AIサービス）
  • 送信経路: アプリ → Firebase Cloud Functions（サーバー） → Google Gemini API
  • ユーザーの音声データが端末から直接 Gemini API に送信されることはなく、当社のサーバー（Firebase Cloud Functions）を経由します

3.3 利用目的と保持期間

  • 送信された音声データは採点処理のみに使用されます
  • Gemini API による採点処理が完了した後、APIに送信された音声データは直ちに破棄されます
  • GoogleのAIモデルの学習データとして使用されることはありません

3.4 同意の取得と管理

  • 初回のAI採点実行時に、データ送信に関する同意ダイアログがアプリ内に表示されます
  • ユーザーが明示的に「同意する」をタップした場合のみ、AI採点が実行されます
  • 同意しない場合、AI採点機能は利用できませんが、その他のアプリ機能には影響しません
  • 一度同意した後も、プロフィール画面の「プライバシー設定」からいつでも同意を撤回できます
  • 同意を撤回した場合、次回のAI採点実行時に再度同意ダイアログが表示されます

4. その他の第三者への情報提供

4.1 Firebase (Google Cloud)

ユーザーデータ、音声ファイル、スコアデータは Google が提供する Firebase プラットフォームに保存されます。

4.2 ランキング公開情報

ランキングに掲載されるニックネームとスコアは、本アプリの全ユーザーに公開されます。

上記以外の第三者に個人情報を提供することはありません。

5. データの保管

  • データは Google Cloud（Firebase）のサーバーに保管されます
  • データは暗号化されて送受信されます（HTTPS通信）
  • アクセス制御により、各ユーザーは自分のデータにのみアクセスできます

6. ユーザーの権利

ユーザーは以下の権利を有します：

  • データの閲覧: アプリ内でご自身のスコア履歴やプロフィール情報を確認できます
  • ニックネームの変更: プロフィール画面からいつでもニックネームを変更できます
  • AI採点の同意管理: プロフィール画面の「プライバシー設定」からAIデータ送信の同意をいつでも撤回できます
  • アカウントの削除: プロフィール画面からアカウントおよび関連データ（音声ファイル、スコア履歴を含む）を完全に削除できます

7. 子供のプライバシー

本アプリは全年齢を対象としていますが、13歳未満のお子様が利用される場合は、保護者の同意の上でご利用ください。

8. プライバシーポリシーの変更

本プライバシーポリシーは予告なく変更される場合があります。変更があった場合は、本ページを更新し、最終更新日を変更します。
