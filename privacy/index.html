どうぶつモノマネ選手権

最終更新日: 2026年2月4日

「どうぶつモノマネ選手権」（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本アプリにおける情報の収集、使用、開示に関する方針を説明します。

1. 収集する情報

1.1 アカウント情報

  • Googleアカウントでログインした場合：表示名、メールアドレス、プロフィール画像URL
  • 匿名ログインの場合：Firebase が生成する匿名ユーザーID
  • ユーザーが設定したニックネーム

1.2 音声データ

  • ユーザーが録音した動物の鳴き真似の音声ファイル（3秒間、AAC形式）
  • 音声ファイルは Firebase Storage に保存されます

1.3 利用データ

  • 採点スコアおよびスコアの内訳
  • AIによる評価コメント
  • プレイ回数、挑戦履歴
  • ランキング情報

2. 情報の使用目的

収集した情報は以下の目的で使用します：

  • AI採点: 録音された音声を Google Gemini API に送信し、動物の鳴き真似の類似度を評価します
  • ランキング表示: 全国ランキングおよび動物別ランキングにスコアとニックネームを表示します
  • SNSシェア: ユーザーの操作により、スコアカード画像をSNSにシェアする機能を提供します
  • サービス改善: アプリの品質向上のために利用状況を分析します

3. 第三者への情報提供

3.1 Google Gemini API

音声の採点のため、録音された音声データを Google の Gemini API に送信します。送信された音声データは採点処理完了後に破棄され、Googleの学習データとしては使用されません。

3.2 Firebase (Google Cloud)

ユーザーデータ、音声ファイル、スコアデータは Google が提供する Firebase プラットフォームに保存されます。

3.3 ランキング公開情報

ランキングに掲載されるニックネームとスコアは、本アプリの全ユーザーに公開されます。

上記以外の第三者に個人情報を提供することはありません。

4. データの保管

  • データは Google Cloud（Firebase）のサーバーに保管されます
  • データは暗号化されて送受信されます（HTTPS通信）
  • アクセス制御により、各ユーザーは自分のデータにのみアクセスできます

5. ユーザーの権利

ユーザーは以下の権利を有します：

  • データの閲覧: アプリ内でご自身のスコア履歴やプロフィール情報を確認できます
  • ニックネームの変更: プロフィール画面からいつでもニックネームを変更できます
  • アカウントの削除: アカウントおよび関連データの削除をリクエストできます
  • 音声データの削除: 録音された音声データの削除をリクエストできます

データの削除をご希望の場合は、下記の連絡先までお問い合わせください。

6. 子供のプライバシー

本アプリは全年齢を対象としていますが、13歳未満のお子様が利用される場合は、保護者の同意の上でご利用ください。

7. プライバシーポリシーの変更

本プライバシーポリシーは予告なく変更される場合があります。変更があった場合は、本ページを更新し、最終更新日を変更します。
